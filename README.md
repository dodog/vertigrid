# VertiGrid

VertiGrid is a GNOME Shell extension that replaces the default horizontal app grid with a vertical app grid in the overview.
It adds category grouping, navigation, smooth scrolling, workspace visibility control, and layout customization for a cleaner overview experience.

## Features

- Vertical app grid in GNOME Shell overview
- Category grouping with navigation sidebar
- Favorites section support
- Favorites can also appear directly in the apps grid
- Drag-and-drop app organization
- Smooth scrolling option
- Toggle workspace bar visibility in the overview
- Customize grid layout:
  - Columns
  - Icon size
  - Icon spacing
  - Category label font size
- Edit category profile from preferences
- Supports GNOME Shell 48, 49, and 50
